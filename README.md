# tindom

A tiny DOM‑first web framework with modular architecture and lightweight reactivity.

tindom focuses on simplicity and minimalism. It works directly with the native DOM, avoids heavy abstractions, and is split into small modules so you only use what you need. It’s designed for small to medium projects that don’t need large, complex frameworks.

---

## Features

- DOM‑first approach — no virtual DOM
- Lightweight reactivity
- Modular design
- TypeScript support
- Small footprint
